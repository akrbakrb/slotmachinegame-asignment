# Unity Slot Machine

## Overview
A simple slot machine game built in Unity.

## Features
- RNG-based reel outcomes
- Smooth reel animations
- Payout system
- Bonus wild symbol
- WebGL build

## Controls
Select Bet Amount to play.

## Build
Open Build/WebGL/index.html

## Technical Design
- OOP architecture
- ReelController
- SlotMachineManager
